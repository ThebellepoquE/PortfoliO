## 🚀 **Getting Started**

```bash
# Clone repository
git clone https://github.com/ThebellepoquE/THEBELLEPOQUE
cd THEBELLEPOQUE

# Install dependencies
npm install

# Start development server
npm start

# Build for production
npm run build
```

**Access:** http://localhost:3000
