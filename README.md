## 🚀 **Quick Start**

```powershell
# Clone and install
git clone https://github.com/ThebellepoquE/thebellepoque-portfolio.git
cd PORTFOLIO/THEBELLEPOQUE
npm install

# Start development
npm start
```

🌐 **Open:** http://localhost:3000

**For production:** `npm run prod`

## ✨ **Key Features**

- **Admin CRUD operations** for all content
